def time_str_to_seconds(time_str):
    time_str = time_str.replace(',', '.')  # 00:00:42,266 → 00:00:42.266
    hh_mm_ss, milliseconds = time_str.split('.')
    hh, mm, ss = hh_mm_ss.split(':')
    total_seconds = (int(hh) * 3600 + int(mm) * 60 + float(f"{ss}.{milliseconds}"))

## 六、关键错误点（核心）

当前代码：

```bash
ffmpeg -i input.wav -ss START -to END
```

问题：

1. ❌ `-ss` 放在输入后 → 切割不精确（会往前吃音频）
2. ❌ 使用 `-to`（绝对时间） → 语义错误

---

## 七、正确方案（必须修改）

```python
duration = end - start

cmd = [
    "ffmpeg",
    "-ss", str(start),          # 放在输入前
    "-i", input_audio_file,
    "-t", str(duration),        # 用 duration，不用 -to
    "-c:a", "pcm_s16le",
    "-y", temp_file
]
```
